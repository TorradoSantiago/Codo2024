# HardSoft

Sitio comercial demo para un negocio de hardware, upgrades y soporte.

## Objetivo

Mostrar una propuesta de venta mas profesional, con home comercial, catalogo filtrable, pagina de productos, contacto y portal demo de clientes.

## Herramientas

- HTML5
- CSS3
- JavaScript vanilla
- Google Fonts
- Assets estaticos para imagenes y video

## Conclusion

El repo ya funciona como una pieza de portfolio mas completa y clara, con base lista para sumar backend, stock o integraciones reales.
